# YOLO Object Detection

## Description
This project implements an object detection system using
the YOLO (You Only Look Once) algorithm to detect and
classify objects in images or videos.

## Technologies Used
- Python
- OpenCV
- YOLO
- TensorFlow / PyTorch

## Dataset
COCO / Custom Dataset

## Key Features
- Real-time object detection
- High accuracy and speed

## Learning Outcome
This project helped me understand computer vision,
deep learning models, and real-time detection systems.
